# Release Notes — v1.2.2

**Date:** 2026-04-06
**Type:** Infrastructure Migration

---

## Summary

Migrated the **Public Server Proxy** deployment target from Render.com / Oracle Cloud to **Google Cloud Platform (GCP)**. The proxy now runs on a GCE `e2-micro` Always Free Tier VM with a dedicated static IP, eliminating both the cold-start delays of Render and the connectivity issues of Oracle.

---

## Changes

### GCP Compute Engine Deployment
- The proxy server is deployed on a **GCE `e2-micro` VM** (1 shared vCPU, 1 GB RAM) in GCP's Always Free Tier.
- A **reserved static external IP** is whitelisted on the CoC Developer Portal — no shared NAT pools, no dynamic IPs.
- The VM is **always on** — zero cold-start delays (unlike Render's 45-second wake-up).
- GCP requires a credit card for identity verification but does NOT auto-charge after free credits expire.
